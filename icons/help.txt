### MacOS icons ### :


1. install 

* git clone https://github.com/vinceliuice/MacTahoe-icon-theme
* cd MacTahoe-icon-theme/
* ./install.sh

2. setup

    aFor KDE-Plasma:
    
from System Settings ---> Global Theme ---> Icons --->   MacTahoe OR MacTahoe-dark OR MacTahoe-light


    b. For Gnome:
    
Tweaks ---> Appearance ---> Icons ---> MacTahoe OR MacTahoe-dark OR MacTahoe-light
